# /data — Personal Data Directory

**WARNING: This directory is git-ignored. Files here will NOT be pushed to the remote repository.**

Place your raw data files here:
- Charger server logs
- MongoDB exports / dumps
- CSV, JSON, Excel files

## Structure

```
data/
├── raw/          # Original, untouched source data
├── processed/    # Cleaned and transformed data
└── datasets/     # ML/AI-ready datasets
```
